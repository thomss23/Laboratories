# Syllabus

1. Basic knowledge of OOP
   * Classes, objects
   * Inheritance
   * Composition
   * Polymorphism
   * Encapsulation
   * Abstraction
 2. Design Patterns
    * Singleton
    * Factory
    * Observer
    * Strategy
    * Command
    * Visitor
    * Chain of Responsability
 3. Programming in Java
    * Basic syntax, data types
    * Abstract classes and interfaces
    * Inner Classes
    * Lambdas
    * Collections and generics
    * I/O and Exceptions
